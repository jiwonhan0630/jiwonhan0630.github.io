## 프로젝트 개요

#### 플랫폼
- PC (Windows)
- 모바일 (Android, iOS)

#### 장르
- 작업 연동 게이미피케이션

#### 개발 환경
- Unity Engine / MariaDB
- Java Script / TMaxTiberoDB

## 담당 업무
- 레벨 제작을 위한 커스텀 에디터 기능 구현
- 자동차 AI (FSM) 구현
- 자동차 부품 시스템 구현
    - 주행 중 자동차 부품의 장착, 변경, 제거 구현
    - 자동차 부품에 따른 스탯 변동 구현
    - 자동차 부품 데이터를 게임 서버 DB와 연동
- Field Section 시스템 구현
- Field Object 시스템 구현
- 3D 사운드 지원을 위한 사운드 기능 구현
