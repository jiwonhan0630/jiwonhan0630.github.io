### 프로젝트 개요
- PC (Windows)
- Unity Engine
- 3D 횡스크롤 슈팅 플랫포머

## 담당 업무

### 보스 몬스터
- 보스 몬스터 FSM: AnimationStateMachine을 사용하여 애니메이션과 실제 상태를 연동하고.,..이벤트도...
- Inspector 기반 보스 패턴 에디터: 코드를 수정하지 않고도 인스펙터 상에서 보스의 공격 패턴, 타이밍, 페이즈 전환 조건 등을 설정할 수 있는 시스템을 구축하여 반복적인 수정 작업 시간을 단축함.
- Gizmos 활용 보스 스테이지 규격화: Gizmos를 통해 스테이지를 가상으로 시각화하고 수정할 수 있음.  
씬에 배치될 리소스에 구애받지 않고 보스의 스테이지 기반 스킬 및 패턴을 디자인할 수 있도록 함.

### 기타
- 스테이지 연출: 시네머신을 사용하여 각 스테이지별 시작 연출 제작
- 오브젝트 풀링 시스템: 템플릿을 통해 간단하게 사용할 수 있는 오브젝트 풀링 스크립트 작성
- Reflection 기반 동적 키 바인딩: 하드코딩된 키 입력을 수정, Reflection을 활용하여 키를 매핑하도록 함
- CSV 파일을 사용한 대사 출력
- UI 시스템
