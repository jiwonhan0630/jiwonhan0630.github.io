## 프로젝트 개요

#### 장르
- 3D 횡스크롤 슈팅 플랫포머

#### 개발 환경
- Unity Engine


## 담당 업무
- GitHub를 사용한 게임 버전 관리
- 보스 시스템 구현
    - 보스 몬스터의 Inspector를 통한 패턴 에디터 기능
    - 보스 몬스터의 페이즈 기능
    - 보스 몬스터 애니메이션 일부
- 보스 몬스터 등장 연출
- 오브젝트 풀 리팩토링
    - 팀 내부에서 사용하던 오브젝트 풀을 개선하여 성능과 작업속도를 향상
- 튜토리얼 스테이지 구현
- 리플렉션을 통한 키세팅 구현
- csv 파일을 사용한 대사 출력
- 사운드
    - 부드러운 사운드 전환 구현
    - 인게임 사운드 리소스 로드 기능 구현
