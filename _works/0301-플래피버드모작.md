# 담당 업무
- 랭킹 시스템
- 개발용 로그 시스템

## 세부 내용

### 랭킹 시스템
MongoDB와 연동하여 유저의 플레이 점수를 저장하고 UI에 출력하는 기능 구현

### 개발용 로그 시스템
Unity 기본 로그를 확장하여 개발에 필요한 정보를 빠르게 출력 및 저장할 수 있는 로그 클래스 구현

### 정규 표현식 공부
- 유저 이름: 불필요한 특수 문자나 공백을 제한하는 유효성 검사 기능 구현
- 로그 파일 저장: 가독성을 저해하는 요소(리치 텍스트 등)을 일괄 제거하여 필요한 부분만 정제하여 저장
